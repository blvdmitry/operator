BUIProvider provides global meta information, like content direction and serves as a global wrapper for the whole application.

```
import { BUIProvider } from '@bookingcom/bui-react';
import LegacyTheme from '@bookingcom/bui-react/themes/legacy';

const App = ({ children }) => (
  <BUIProvider theme={LegacyTheme}>
    { children }
  </BUIProvider>
);
```

Anywhere inside the provider - you can get access to the BUI React hooks.

### Properties

| Name                | Type                                 | Default | Required | Description                              |
| ------------------- | ------------------------------------ | ------- | -------- | ---------------------------------------- |
| children            | `ReactNode`                          | -       | false    | Provider children elements               |
| theme               | `Theme`                              | -       | false    | Theme component                          |
| themeMode           | `light`, `dark`                      | false   |          | Controlled color mod                     |
| id                  | `string`                             | -       | false    |                                          |
| defaultThemeMode    | `light`, `dark`                      | -       | false    | Default theming mode for the application |
| defaultRTL          | `boolean`                            | -       | false    | Default RTL flag value for SSR           |
| defaultViewportSize | `small`, `medium`, `large`, `xlarge` | `small` | false    | Default Viewport Size for SSR            |
