## Usage

```jsx
import { BottomNavigation } from "@bookingcom/bui-react";
import type { BottomNavigationProps } from "@bookingcom/bui-react";
```

A Bottom Navigation component can have multiple `items` which are in most cases an Icon. Each of the items can also have a Bubble to signify a new notification by using the `notificationValue` prop.

<Example name="default" />

It is also possible for an item to contain an Avatar instead of an Icon. Design guidelines sugges that an Avatar should be only the last item.

<Example name="avatar" />

Every item can have an `id` that is used to detrmine which item is selected by using the `selectedId` prop. Design guidelines propose that bold icons be used for selected items.

## Accessibility

- If you have to use bottom navigation without labels:
  - Work with a copywriter to come up with a suitable and concise label that describes each icon. Make sure they are available via `alt` or `aria-label` attributes.
- If showing notification bubbles:
  - Unless the context is clear, include additional context with a meaningful `aria-label`, that describes what the content of the bubble relates to and if the content requires immediate attention and action.
- Use bold icons for selected tabs, so itâ€™s easier to navigate for colour blind users.

## Properties

| Name                    | Type                | Default | Required | Description                                                                                                    |
| ----------------------- | ------------------- | ------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| items                   | `array`             | -       | true     | Accordion items                                                                                                |
| items.id                | `string`, `number`  | -       | false    | Item id                                                                                                        |
| items.icon              | `ReactNode`         | -       | false    | Item icon svg (either of Icon or Avatar are required)                                                          |
| items.avatar            | `AvatarProps`       | -       | false    | Avatar props (either of Avatar or Icon are required)                                                           |
| items.text              | `ReactNode`         | -       | false    | Item text                                                                                                      |
| items.notificationValue | `number`, `boolean` | -       | false    | Item notification bubble value. Use boolean for displaying an empty bubble                                     |
| items.onChoose          | `(item, e) => void` | -       | false    | Item choose handler                                                                                            |
| items.href              | `string`            | -       | false    | Item link href                                                                                                 |
| items.attributes        | `object`            | -       | false    | Custom item element attributes                                                                                 |
| items.preventDefault    | `boolean`           | -       | false    | Should prevent event's default behaviour?                                                                      |
| selectedId              | `string`, `number`  | -       | true     | Id of the selected item                                                                                        |
| className               | `string`            | -       | false    | Custom root element className                                                                                  |
| attributes              | `object`            | -       | false    | Custom root element attributes                                                                                 |
| mixin                     | `object`            | -       | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |
