# Box

<Featured>Use box to add standard padding to the content.</Featured>

## Usage

```jsx
import { Box } from "@bookingcom/bui-react";
import type { BoxProps } from "@bookingcom/bui-react";
```

This utility is especially handy combined will fill variants of the components when text content has to be used together with full-width content.

It accepts variuos `padding` values (background is for readability):

<Example name="padding" />

With `orientation` prop you can apply padding only on a certain axis.

<Example name="orientation" />

In combination with other `borderRadius`, `borderColor` and `backgroundColor` props, Box becomes very flexibile utility.

<Example name="other" />

## Responsive properties

Padding property can be responsive. That means it can have different value between viewports. You can read more about responsive properties [here](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#Responsive-development).

## Properties

| Name            | Type                                                                                                                                                                        | Default | Required | Description                                                                                                                               |
| --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| children        | `ReactNode`                                                                                                                                                                 | -       | false    | Box content                                                                                                                               |
| padding         | `number`                                                                                                                                                                    | `4`     | false    | [Responsive](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#Responsive-development). Box padding size preset |
| orientation     | `horizontal`, `vertical`                                                                                                                                                    | -       | false    | Box padding orientation preset                                                                                                            |
| borderRadius    | `100`, `200`, `300`, `circle`                                                                                                                                               | -       | false    |
| borderColor     | `destructive`, `constructive`, `accent`, `callout`, `neutral`, `neutral_alt`, `action`                                                                                      | -       | false    |
| backgroundColor | `destructive_alt`, `constructive_alt`, `accent_alt`, `action_alt`, `callout_alt`, `neutral_alt`, `brand_primary`, `brand_primary_dynamic`, `elevation_one`, `elevation_two` | -       | false    |
| overflow        | `hidden`                                                                                                                                                                    | -       | false    |
| className       | `string`                                                                                                                                                                    | -       | false    | Custom root element className                                                                                                             |
| attributes      | `object`                                                                                                                                                                    | -       | false    | Custom root element attributes                                                                                                            |
| mixin             | `object`                                                                                                                                                                    | -       | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer)                            |
