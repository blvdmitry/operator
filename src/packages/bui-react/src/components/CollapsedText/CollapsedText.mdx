## Usage

```jsx
import { CollapsedText } from "@bookingcom/bui-react";
import type { CollapsedTextProps } from "@bookingcom/bui-react";
```

All you need to render basic version of CollapsedText is `text`, `readMoreLabel` and `readLessLabel` props.

<Example name="default" />

Specify the text variant via `variant` prop. Notice the button size is changed as well.

<Example name="body1" />

By default, collapsed text has 4 visible lines. Customize it with `visibleLines` prop.

<Example name="visibleLines" />

In case the text is too small to be collapsed, it will be rendered as it is, without the button and gradient. It will also re-adjust on container size change — notice how it is collapsed on mobile viewport and non-collapsed on desktop.

<Example name="tooSmallToCollapse" />

Be aware that CollapsedText works only with plain text passed as string. You can split text in paragraphs by using normal whitespace line-breaks or `\n`.

<Example name="paragraphs" />

## Properties

| Name          | Type               | Default  | Required | Description                                                                                                    |
| ------------- | ------------------ | -------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| text          | `string`           | -        | true     | Text to collapse.                                                                                              |
| variant       | `body_2`, `body_1` | `body_2` | false    | Variant of the text                                                                                            |
| visibleLines  | `number`           | `4`      | false    | Number of lines visible in collapsed state                                                                     |
| readMoreLabel | `string`           | -        | true     | Button text in collapsed state                                                                                 |
| readLessLabel | `string`           | -        | true     | Button text in uncollapsed state                                                                               |
| className     | `string`           | -        | false    | Custom root element className                                                                                  |
| attributes    | `object`           | -        | false    | Custom root element attributes                                                                                 |
| mixin           | `object`           | -        | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |
