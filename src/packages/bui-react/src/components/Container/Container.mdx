# Container

<Featured>
  Container utility component is mainly used to control the width and alignment
  of the main content on page.
</Featured>

## Usage

```jsx
import { Container } from "@bookingcom/bui-react";
import type { ContainerProps } from "@bookingcom/bui-react";
```

<Example name="default" />

Pass `centered` prop to center the container within the parent (should have viewport larger than 1024px).

<Example name="centered" />

## Properties

| Name       | Type        | Default | Required | Description                                                                                                    |
| ---------- | ----------- | ------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| children   | `ReactNode` | -       | true     | Container content                                                                                              |
| centered   | `boolean`   | -       | false    | Should be centered on desktop?                                                                                 |
| className  | `string`    | -       | false    | Custom root element className                                                                                  |
| attributes | `object`    | -       | false    | Custom root element attributes                                                                                 |
| mixin        | `object`    | -       | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |
