## Usage

```jsx
import { DateRange } from "@bookingcom/bui-react";
import type { DateRangeProps } from "@bookingcom/bui-react";
```

DateRange comes in two variant, `simplified` and `detailed`.
`Simplified` variant is used to display dates for specific days without any additional information.
In this case we align all the content to the center of the component.

<Example name="simplified" />

`Detailed` variant aligns everything to the start side and supports rendering additional properties, like `label` and `originalTitle`.
It's used to display more specific date values, including daytime.

<Example name="detailed" />

DateRange items can be rendered as actionable elements whenever you need to either navigate to another page or open a sheet with additional information.

## Accessibility

When date is spread over title and subtitle fill the item `datetime` attribute with the full date so screenreaders can pronounce it correctly.

## Properties

| Name       | Type                     | Default      | Required | Description                                                                                                    |
| ---------- | ------------------------ | ------------ | -------- | -------------------------------------------------------------------------------------------------------------- |
| variant    | `simplified`, `detailed` | `simplified` |          |                                                                                                                |
| from       | `DateItemProps`          | -            | true     |                                                                                                                |
| to         | `DateItemProps`          | -            | true     |                                                                                                                |
| className  | `string`                 | -            | false    | Custom root element className                                                                                  |
| attributes | `object`                 | -            | false    | Custom root element attributes                                                                                 |
| mixin        | `object`                 | -            | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |
