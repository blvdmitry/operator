## Usage

```jsx
import { DismissibleContainer } from "@bookingcom/bui-react";
import type { DismissibleContainerProps } from "@bookingcom/bui-react";
```

Dismissible container is used to wrap any toggleable content and provide a layout for it. By default it does not bring any Javascript functionality because in some cases you don't want to hide elements immediately or need to provide custom handler in addition to toggling. Therefore to add JS logic to the component - you can specify handlers for the close button click event.

## Accessibility

Close button should have an aria-label on it describing its action purpose.

## Properties

| Name           | Type         | Default | Required | Description                            |
| -------------- | ------------ | ------- | -------- | -------------------------------------- |
| children       | `ReactNode`  | -       | false    | Area overlapping with the close button |
| onClose        | `() => void` | -       | false    | Close handler                          |
| closeAriaLabel | `string`     | -       | false    | aria-label attribute for close button  |
| fill           | `boolean`    | -       | false    |
| hideClose      | `boolean`    | -       | false    | Hide the close button?                 |
| className      | `string`     | -       | false    | Custom root element className          |
| attributes     | `object`     | -       | false    | Custom root element attributes         |
| closeClassName | `string`     | -       | false    | Custom close button className          |
