## Usage

```jsx
import { DropdownMenu } from "@bookingcom/bui-react";
import type { DropdownMenuProps } from "@bookingcom/bui-react";
```

Since popovers depend on the coordinates of underlying element - it needs a way to hook into its children event handlers.
In order to solve that issue, we're using a render props pattern for rendering inner components:

```
<DropdownMenu items={[...]}>
  { attributes => <Button text="Open menu" attributes={attributes} />}
</Popover>
```

## Properties

| Name                | Type                                                                                                    | Default        | Required | Description                                    |
| ------------------- | ------------------------------------------------------------------------------------------------------- | -------------- | -------- | ---------------------------------------------- |
| id                  | `string`                                                                                                | -              | false    | Custom id override                             |
| position            | `bottom-end`, `bottom`, `bottom-start`, `bottom-stretch`, `top-end`, `top`, `top-start`, `start`, `end` | `bottom-start` | false    | Dropdown position                              |
| children            | `(attributes) => ReactNode`                                                                             | -              | false    | Dropdown menu trigger content                  |
| onItemChoose        | `(item, index) => void`                                                                                 | -              | false    | Dropdown menu item choose handler              |
| onOpen              | `() => void`                                                                                            | -              | false    | Open handler                                   |
| onClose             | `() => void`                                                                                            | -              | false    | Close handler                                  |
| items               | `array`                                                                                                 | -              | false    | Dropdown menu items                            |
| item.text           | `string`                                                                                                | -              | true     | Item text                                      |
| item.icon           | `ReactNode`                                                                                             | -              | false    | Item icon                                      |
| item.onChoose       | `(item) => void`                                                                                        | -              | false    | Item choose handler                            |
| item.href           | `string`                                                                                                | -              | false    | Item href attribute                            |
| item.disabled       | `boolean`                                                                                               | -              | false    | Is dropdown menu item disabled?                |
| item.textSlot       | `ReactNode`                                                                                             | -              | false    | Item content next to the text                  |
| item.startSlot      | `ReactNode`                                                                                             | -              | false    | Item content at the start of the item          |
| item.endSlot        | `ReactNode`                                                                                             | -              | false    | Item content at the end of the item            |
| item.className      | `string`                                                                                                | -              | false    | Custom dropdown menu item element className    |
| item.attributes     | `object`                                                                                                | -              | false    | Custom dropdown menu item element attributes   |
| item.preventDefault | `boolean`                                                                                               | -              | false    | Should prevent event default?                  |
| sections            | `array`                                                                                                 | -              | false    | Dropdown menu sections                         |
| sections.items      | `array`                                                                                                 | -              | false    | Dropdown menu section items                    |
| navigationMode      | `arrows`, `tab-and-arrows`                                                                              | `arrows`       | false    | Keyboard navigation mode                       |
| disableAnimation    | `boolean`, `hide`                                                                                       | false          | false    | Disable content fade-in and fade-out animation |
| triggerDisplay      | `block`                                                                                                 | false          | false    | Display trigger as block element?              |
| triggerClassName    | `string`                                                                                                | false          | false    | Custom trigger element className               |
| triggerType         | `click`, `focus`                                                                                        | `click`        | false    | Event causing dropdown menu to trigger         |
| attributes          | `object`                                                                                                | false          | false    | Custom root element attributes                 |
