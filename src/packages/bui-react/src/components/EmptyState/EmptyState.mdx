# EmptyState

<Featured>
  Empty states occur when there's no content to show to the user. They are meant
  to provide reassurance and contextual guidance through friendly tone of voice
  and clear actions.

## Usage

Use the empty state component as a placeholder when there is no content available for that area.
If the user can take an action to create content to show, you can add a button compontent that will initiate that action.

```jsx
import { EmptyState } from "@bookingcom/bui-react";
import type { EmptyStateProps } from "@bookingcom/bui-react";
```

Use icons to illustrate the context via `icon` prop.

<Example name="icon" />

Illustrations can be added via `topIllustration` or `startIllustration` props.

<Example name="topIllustration" />

When using `startIllustration` layout is transformed into horizontal (`md` breakpoint and above).

## Accessibility

When using the optional button component, the button text should be as clear as possible. In cases where that might be tough or tricky, then add an `aria-label` attribute to provide context and further details.

## Properties

| Name              | Type            | Default | Required | Description                                                                                                    |
| ----------------- | --------------- | ------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| icon              | `IconComponent` | -       | false    | Empty state icon                                                                                               |
| topIllustration   | `ReactNode`     | -       | false    |
| startIllustration | `ReactNode`     | -       | false    | Switched EmptyState to horizontal mode                                                                         |
| title             | `string`        | -       | false    | Empty state title                                                                                              |
| text              | `string`        | -       | true     | Empty state text                                                                                               |
| button            | `ButtonProps`   | -       | false    | Empty state button                                                                                             |
| link              | `LinkProps`     | -       | false    | Empty state link                                                                                               |
| className         | `string`        | -       | false    | Custom root element className                                                                                  |
| attributes        | `object`        | -       | false    | Custom root element attributes                                                                                 |
| mixin               | `object`        | -       | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |
