## Usage

```jsx
import { InputCheckboxGroup } from "@bookingcom/bui-react";
import type { InputCheckboxGroupProps } from "@bookingcom/bui-react";
```

`InputCheckboxGroup` works with the checkbox fields passed to it as children.
You can control their layout using BUI React utilities, like `Stack`.

<Example name="defaultInline" />

### State

Provide error message via `error` prop to display the message and highlight checkboxes with accent color.
Restrict user from interacting with the interface with `disabled` prop.

<Example name="errorDisabled" />

### Controlled / Uncontrolled

`InputCheckboxGroup` can be controlled and uncontrolled, when it's using `defaultValue` and `value` props respectively.
By making component controlled, you fully control the event handling.

<Example name="selectedControlled" />

### Composition

You can use `InputCheckboxGroup` with other checkbox components as well and apply custom layout to them.
For example, you can pass `InputCheckboxCard` or `Chip` as children.

<Example name="card" />
