## Usage

```jsx
import { InputRadio } from "@bookingcom/bui-react";
import type { InputRadioProps } from "@bookingcom/bui-react";
```

## Controlled and uncontrolled

All input components in BUI support controlled and uncontrolled version.
Uncontrolled version is the default behavior where you can pass it a `defaultChecked` and the browser will control the InputRadio's checked or unchecked state.

<Example name="uncontrolledChecked" />

Controlled version is enabled once you pass the `checked` prop to InputRadio.
This means that clicking on the InputRadio will trigger `onChange` but you will have to update the `checked` prop using your own state.
