## Usage

```jsx
import { InputRadioGroup } from "@bookingcom/bui-react";
import type { InputRadioGroupProps } from "@bookingcom/bui-react";
```

`InputRadioGroup` works with the radio fields passed to it as children.
You can control their layout using BUI React utilities, like `Stack`.

<Example name="default" />

### State

Provide error message via `error` prop to display the message and highlight radio inputs with destructive color.

<Example name="error" />

Restrict user from interacting with the interface with `disabled` prop.

<Example name="disabled" />

### Controlled / Uncontrolled

`InputRadioGroup` can be controlled and uncontrolled, when it's using `defaultValue` and `value` props respectively.
By making component controlled, you fully control the event handling.

<Example name="selectedControlled" />

### Composition

You can use `InputRadioGroup` with other radio components as well and apply custom layout to them.
For example, you can pass `InputRadioCard` as children.

<Example name="card" />
