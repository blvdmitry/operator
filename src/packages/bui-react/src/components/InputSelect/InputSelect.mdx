# InputSelect

<Featured>
  Input select allows users to choose one option from a list. Interacting with
  them displays toggleable, contextual overlays for displaying lists of options.
  As such, they can provide a more economical use of space than radio buttons.
</Featured>

## Usage

```jsx
import { InputSelect } from "@bookingcom/bui-react";
import type { InputSelectProps } from "@bookingcom/bui-react";
```

## Properties

| Name               | Type                        | Default | Required | Description                                                                                                    |
| ------------------ | --------------------------- | ------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| label              | `ReactNode`                 | -       | false    | Field label                                                                                                    |
| subLabel           | `ReactNode`                 | -       | false    | Field sub label                                                                                                |
| id                 | `string`                    | -       | false    | Custom field id                                                                                                |
| value              | `string`, `null`            | -       | false    | Field value. Activates controlled version of the component                                                     |
| defaultValue       | `string`                    | -       | false    | Field default value. Activates uncontrolled version of the component                                           |
| name               | `string`                    | -       | true     | Field name                                                                                                     |
| size               | `large`                     | -       | false    | Field size                                                                                                     |
| startIcon          | `IconComponent`             | -       | false    |                                                                                                                |
| startAvatar        | `AvatarProps`               | -       | false    | Avatar properties                                                                                              |
| startImage         | `ImageProps`                | -       | false    | Image properties                                                                                               |
| onChange           | `({ name, value }) => void` | -       | false    | Field change handler                                                                                           |
| onFocus            | `(event) => void`           | -       | false    | Field focus handler                                                                                            |
| onBlur             | `(event) => void`           | -       | false    | Field blur handler                                                                                             |
| onClick            | `(event) => void`           | -       | false    | Field click handler                                                                                            |
| placeholder        | `string`                    | -       | false    | Field placeholder                                                                                              |
| options            | `array`                     | -       | false    | Field options. When niether `options` of `optgroups` defined, the component renders as a `<Button ... />`      |
| options.text       | `string`                    | -       | true     | Option text                                                                                                    |
| options.value      | `string`                    | -       | true     | Option value                                                                                                   |
| optgroups          | `array`                     | -       | false    | Group of options. When niether `options` of `optgroups` defined, the component renders as a `<Button ... />`   |
| optgroups.label    | `string`                    | -       | true     | Option group label                                                                                             |
| optgroups.disabled | `boolean`                   | -       | false    | If option group is disabled                                                                                    |
| error              | `string`                    | -       | false    | Field error text                                                                                               |
| helper             | `string`                    | -       | false    | Field helper text                                                                                              |
| success            | `string`                    | -       | false    | Field success text                                                                                             |
| disabled           | `boolean`                   | -       | false    | Is field disabled?                                                                                             |
| required           | `boolean`                   | `false` | false    | Mark the input field as required in the label                                                                  |
| bordered           | `boolean`                   | `true`  | false    | Render the borders around the select                                                                           |
| className          | `string`                    | -       | false    | Custom root element className                                                                                  |
| inputClassName     | `string`                    | -       | false    | Custom select element className                                                                                |
| attributes         | `object`                    | -       | false    | Custom root element attributes                                                                                 |
| inputAttributes    | `object`                    | -       | false    | Custom select element attributes                                                                               |
| children           | `ReactNode`                 | -       | false    | Content for select when it's used as a trigger without options                                                 |
| mixin                | `object`                    | -       | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |
