## Usage

```jsx
import { InputText } from "@bookingcom/bui-react";
import type { InputTextProps } from "@bookingcom/bui-react";
```

<Example name="default" />

Provide `error`, `success` or `helper` props to display error/success/helper texts accordingly.

<Example name="status" />

Make it non-interactable with `disabled` prop.

<Example name="disabled" />

User `prefix` and `suffix` props to provide additional info about the value type of the input (for example, currency).

<Example name="prefixSuffix" />

In similar way you with `startSlot` and `endSlot` you can add icons or other conent to the start and end of the input.

<Example name="startIcon" />

Show `counter` and limit the input with `maximumLength`.

<Example name="maxLength" />

You can enable an additional clear button displayed for the input with a `showClearButton`.
By default it's always hidden and there are 2 support behaviours.

When used with `on-edit` value, clear button will be shown only when the input is focused and its value is not empty.
When used with `always` value, clear button stays visible even after you move the focus away from the input while its value is not empty.

<Example name="clearButton" />

You can provide additional sub-label using `subLabel` prop.
One of the common use-cases for it is to mark field as `(optional)`.

<Example name="sublabel" />

## Properties

| Name               | Type                         | Default  | Required | Description                                                                                                                  |
| ------------------ | ---------------------------- | -------- | -------- | ---------------------------------------------------------------------------------------------------------------------------- |
| id                 | `string`                     | -        | false    | Custom field id                                                                                                              |
| label              | `ReactNode`                  | -        | false    | Field label                                                                                                                  |
| subLabel           | `ReactNode`                  | `false ` | false    | Field sub-label                                                                                                              |
| value              | `string`                     | -        | false    | Field value. Activates the controlled version of the component                                                               |
| defaultValue       | `string`                     | -        | false    | Field default value. Activates the uncontrolled version of the component                                                     |
| type               | `string`                     | `text`   | false    | Field type attribute                                                                                                         |
| name               | `string`                     | -        | true     | Field name                                                                                                                   |
| size               | `medium`, `large`            | `medium` | false    | Field size                                                                                                                   |
| disabled           | `boolean`                    | -        | false    | Is field disabled?                                                                                                           |
| error              | `string`, `boolean`          | -        | false    | Field error text                                                                                                             |
| helper             | `string`                     | -        | false    | Field helper text                                                                                                            |
| success            | `string`                     | -        | false    | Field success text                                                                                                           |
| placeholder        | `string`                     | -        | false    | Field placeholder                                                                                                            |
| prefix             | `string`                     | -        | false    | Field prefix (It is recommended to use either `prefix` or `suffix` and not both)                                             |
| suffix             | `string`                     | -        | false    | Field suffix (It is recommended to use either `prefix` or `suffix` and not both)                                             |
| startSlot          | `ReactNode`                  | -        | false    | slot for custom content at the start of the input                                                                            |
| endSlot            | `ReactNode`                  | -        | false    | slot for custom content at the end of the input                                                                              |
| maximumLength      | `number`                     | -        | false    | Maximum number of characters allowed in the input                                                                            |
| showLengthCounter  | `boolean`                    | false    | false    | Show the current character count of the input, along with the maximum allowed length of the input (based on `maximumLength`) |
| showClearButton    | `never`, `always`, `on-edit` | `never`  | false    | Show clear button which clears the input. The button is shown when the input value not empty                                 |
| required           | `boolean`                    | false    | false    | Mark the input field as required in the label                                                                                |
| bordered           | `boolean`                    | true     | false    | Render the borders around the input                                                                                          |
| onChange           | `({ name, value }) => void`  | -        | false    | Field change handler                                                                                                         |
| onFocus            | `(event) => void`            | -        | false    | Field focus handler                                                                                                          |
| onBlur             | `(event) => void`            | -        | false    | Field blur handler                                                                                                           |
| className          | `string`                     | -        | false    | Custom root element className                                                                                                |
| inputClassName     | `string`                     | -        | false    | Custom input element className                                                                                               |
| decoratorClassName | `string`                     | -        | false    | Custom decorator element className                                                                                           |
| attributes         | `object`                     | -        | false    | Custom root element attributes                                                                                               |
| inputAttributes    | `object`                     | -        | false    | Custom input element attributes                                                                                              |
| prefixAttributes   | `object`                     | -        | false    | Custom prefix element attributes                                                                                             |
| suffixAttributes   | `object`                     | -        | false    | Custom suffix element attributes                                                                                             |
| mixin                | `object`                     | -        | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer)               |
