# ListItem

<Featured>
  List item containers are used in lists. It is an actionable container that
  allows for user interaction such as hover and active states.
</Featured>

## Usage

```jsx
import { ListItem } from "@bookingcom/bui-react";
import type { ListItemProps } from "@bookingcom/bui-react";
```

We can choose the `spacing` between the `startSlot`, `endSlot` and the text content of the ListItem.
The `spacing` prop takes three values: `small, medium, large`, the default being `medium`.

<Example name="spacing" />

Having lists with some selected items becomes possible by passing an `active` prop.

<Example name="active" />

And similarly, we could also disable the ListItem

<Example name="disabled" />

When we do not want a ListItem to be interactive, we omit the `onClick` and `href` props, this removes the hover and active states from the ListItem.
Use `edgeSpacing` property to keep padding in non-interactive variants.

<Example name="nonInteractive" />

We can build complex ListItems using `startSlot` and `endSlot` props by passing different components to each of them as well.

## Properties

| Name              | Type                       | Default  | Required | Description                                                                                                    |
| ----------------- | -------------------------- | -------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| children          | `ReactNode`                | -        | false    | Content                                                                                                        |
| startSlot         | `ReactNode`                | -        | false    | Start slot content                                                                                             |
| endSlot           | `ReactNode`                | -        | false    | End slot content                                                                                               |
| icon              | `ReactNode`                | -        | false    | List item icon, replaces startSlot content                                                                     |
| spacing           | `small`, `medium`, `large` | `medium` | false    | List item spacing (replaces `size` prop)                                                                       |
| verticalAlignment | `start`, `center`          | `center` | false    | Vertically align the contents                                                                                  |
| edgeSpacing       | `boolean`                  | false    | false    | Enables padding for non-interactive variant                                                                    |
| href              | `string`                   | -        | false    | Link href                                                                                                      |
| roundedCorners    | `boolean`                  | true     | false    | Adds rounded corners to the list item                                                                          |
| active            | `boolean`                  | -        | false    | Is list item active?                                                                                           |
| disabled          | `boolean`                  | -        | false    | Is list item disabled?                                                                                         |
| preventDefault    | `boolean`                  | -        | false    | Should prevent event default?                                                                                  |
| className         | `string`                   | -        | false    | Custom root element className                                                                                  |
| attributes        | `object`                   | -        | false    | Custom root element attributes                                                                                 |
| mixin               | `object`                   | -        | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |
