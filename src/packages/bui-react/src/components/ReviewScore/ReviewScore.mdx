## Usage

Use review-score to display user generated review scores.

```jsx
import { ReviewScore } from "@bookingcom/bui-react";
import type { ReviewScoreProps } from "@bookingcom/bui-react";
```

Use `variant` to control ReviewScore appearence.

Control alignment via `alignment` prop.

<Example name="alignment" />

For more simplified look, use `inline` prop, or don't include `ratingReviewAriaLabel` and `rating` props to display rating as a badge.

## Accessibility

Make sure to set the aria-label fields with a copy tag. Below are some recommended copy tags you can use for this component.

- Score Rating: `a11y_rating_score_for_screenreader`
- Score Word: `a11y_adjective_rating`
- Total Number of Reviews: `a11y_comp_num_reviews`.
