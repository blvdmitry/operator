## Accessibility

In case niether `labelStart` nor `labelEnd` is provided, make sure to pass `ariaLabel` property to provide additional element description to screen-readers.

## Properties

| Name       | Type                                                                          | Default | Required | Description                                                                                                    |
| ---------- | ----------------------------------------------------------------------------- | ------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| variant    | `brand_primary`, `constructive`, `accent`, `callout`, `destructive`, `action` | -       | false    |                                                                                                                |
| labelStart | `string`                                                                      | -       | false    | Score bar label                                                                                                |
| labelEnd   | `string`                                                                      | -       | false    | Custom label to render instead of the score value                                                              |
| value      | `string`                                                                      | -       | true     | Score value                                                                                                    |
| ariaLabel  | `string`                                                                      | -       | false    | a11y label                                                                                                     |
| className  | `string`                                                                      | -       | false    | Custom root element className                                                                                  |
| attributes | `object`                                                                      | -       | false    | Custom root element attributes                                                                                 |
| mixin        | `object`                                                                      | -       | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |
