## Usage

```jsx
import { SegmentedControl } from "@bookingcom/bui-react";
import type { SegmentedControlProps } from "@bookingcom/bui-react";
```

Segmented control renders the passed options as buttons that provide radio-like behavior.

<Example name="default" />

When used on mobile or inside containers with limited width, you can use `fillEqually` to have SegmentedControl take full width of its parent component.
That will also stretch its items to have the same width.

<Example name="fillEqually" />

## Accessibility

The segmented control label, that is read only, should reflect the purpose of the selection, eg. 'select your view type'.

## Properties

| Name          | Type                        | Default | Required | Description                                                                                                    |
| ------------- | --------------------------- | ------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| id            | `string`                    | -       | false    | Custom id override                                                                                             |
| label         | `string`                    | -       | false    |
| name          | `string`                    | -       | true     |
| value         | `string`, `null`            | -       | false    | Turns on the controlled mode                                                                                   |
| defaultValue  | `string`                    | -       | false    | Keeps uncontrolled mode                                                                                        |
| options       | `array`                     | -       | false    |
| options.text  | `string`                    | -       | true     |
| options.value | `string`                    | -       | true     |
| onChange      | `({ name, value }) => void` | -       | true     |
| fillEqually   | `boolean`                   | -       | false    |
| className     | `string`                    | -       | false    | Custom root element className                                                                                  |
| mixin           | `object`                    | -       | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |
