## Usage

```jsx
import { SkeletonLoader } from "@bookingcom/bui-react";
import type { SkeletonLoaderProps } from "@bookingcom/bui-react";
```

There are a few variants of the Skeleton available to be used as a placeholder of text elements.
For headings or titles, the `title` variant can be used.

<Example name="title" />

For a single line of text, the `one-line` variant can be used. For multiline text, the `two-lines` or `three-lines` variants can be used. The `three-lines` variant is mostly used for paragraphs.

<Example name="oneLine" />

For colored backgrounds, use `color="inherit"` and set CSS `color` property with any on-background color yourself.

<Example name="inherit" />

The `box` variant can be used to show skeleton loaders for images or other media content.
Defining the `width` and `aspectRatio` props will automatically fix the height of the box.

<Example name="box" />

We can componse Skeleton components with Stack or similar components to construct different card or page layouts.

## Accessibility

Add `aria-busy="true"` to the wrapper node to indicate to screenreaders that the elements within the wrapper are being modified and should not be exposed to the user. When all content is loaded make sure to set aria-busy to `false`.

To announce content when loaded add `aria-live="polite"` to the SkeletonLoader wrapper node.

## Properties

| Name        | Type                                                   | Default   | Required | Description                                                                                                    |
| ----------- | ------------------------------------------------------ | --------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| variant     | `title`, `one-line`, `two-lines`, `three-lines`, `box` | -         | true     | Different variants available to build different types of skeleton cards                                        |
| color       | `neutral`, `inherit`                                   | `neutral` | false    | Color of the skeleton, for colored backgrounds use `inherit` and apply CSS `color` property yourself.          |
| width       | `string`                                               | `100%`    | false    | Width of the `box` variant. Use with `aspectRatio` prop to define the height of the skeleton box.              |
| aspectRatio | `string`                                               | `1:1`     | false    | Aspect ratio of the `box` variant. Use with the `width` property                                               |
| tagName     | `string`                                               | -         | false    | Custom root element tag                                                                                        |
| className   | `string`                                               | -         | false    | Custom root element className                                                                                  |
| attributes  | `object`                                               | -         | false    | Custom root element attributes                                                                                 |
| mixin         | `object`                                               | -         | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |
