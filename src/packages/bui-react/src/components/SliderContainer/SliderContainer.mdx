## Usage

```jsx
import { SliderContainer } from "@bookingcom/bui-react";
import type { SliderContainerProps } from "@bookingcom/bui-react";
```

SliderContainer expects its children to only contain `SliderContainer.Item` components and comes in two variants.

`media` variant is used for display media content like photos.
In this variant, navigation controls get displayed on top of the content to navigate through the items.

<Example name="media" />

In case you need sliding behaviour for other types of content, SliderContainer supports `content` variant.
This removes navigation controls and turns on controlled behaviour for scrolling.
This means that you can use its `ref` to get access to the navigation methods, like `ref.navigate`, `ref.navigateForward` and `ref.navigateBack`.

Here is an example of a slider, that has navigation buttons inside its items:

<Example name="content" />

By default, SliderContainer supports infinite scrolling of the items but you can disable this behaviour with `infinite={false}`.

<Example name="finite" />

SliderContainer has its controls hidden originally and shown only on hover.
In case you need to make them visible all the time, use `showNavigationControls` property.

<Example name="showNavigation" />

## Accessibility

- Do not autoplay the slider

## Properties

| Name                     | Type                     | Default                                                   | Required | Description                                      |
| ------------------------ | ------------------------ | --------------------------------------------------------- | -------- | ------------------------------------------------ |
| variant                  | `media`, `content`       | `media`                                                   | false    | SliderContainer variant                          |
| infinite                 | `boolean`                | `true` for `media` variant, `false` for `content` variant | false    | Activates SliderContainer infinite scrolling     |
| showNavigationControls   | `boolean`                | `false`                                                   | false    | SliderContainer variant                          |
| isScrollEnabled          | `boolean`                | `true`                                                    | false    | Disable scrolling if needed                      |
| children                 | `SliderContainer.Item[]` | -                                                         | false    | SliderContainer content                          |
| defaultActiveIndex       | `number`                 | -                                                         | false    | Index of the item shown by default               |
| previousButtonAriaLabel  | `string`                 | -                                                         | true     | Accessibility label for the previous item button |
| nextButtonAriaLabel      | `string`                 | -                                                         | true     | Accessibility label for the previous item button |
| onAfterNavigate          | `({ index }) => void`    | -                                                         | false    | Navigation handler                               |
| onNavigationControlClick | `(event) => void`        | -                                                         | false    | On control click handler                         |
| borderRadius             | `100`, `200`, `300`      | -                                                         | false    | Border radius of the container                   |
| className                | `string`                 | -                                                         | false    | Custom root element className                    |
| attributes               | `object`                 | -                                                         | false    | Custom root element attributes                   |

### Item properties

| Name     | Type        | Default | Required | Description            |
| -------- | ----------- | ------- | -------- | ---------------------- |
| children | `ReactNode` | -       | false    | Media slider content   |
| src      | `string`    | -       | false    | Url for the item image |

### Ref

You can access SliderContainer imperative methods by passing a `ref` to it:

- `ref.navigate(index)` - scroll Carousel to the passed item index
- `ref.navigateBack()` - scroll Carousel to the passed item index
- `ref.navigateForward()` - scroll Carousel to the passed item index
