# Tab

<Featured>Tabs enable content organization at a high level, such as switching between different sets of content in the same space. The tab bar is presented as a single row above its associated content.</Featured>

## Usage

```jsx
import { Tab } from "@bookingcom/bui-react";
import type { TabProps } from "@bookingcom/bui-react";
```

By default, Tab is using `underlined` variant.

<Example name="default" />

But it also supports `rounded` variant

<Example name="rounded" />

We can also add Icons and Bubbles to the Tab Triggers. Icons provide a visual cue for what the content is going to be. The bubble can be used to notify the user of the number of items in the content. For example, we can use a bubble to notify the user of unread messages in a tab.

<Example name="iconsBubbles" />

The Triggers can be stretched to make them have the same width by using the `fillEqually` prop. This is useful in cases where the Tabs are inside containers and need to take up all available width.

<Example name="equallySized" />

Even though we recommend to have a copy that fits into a single line, there are some languages that copy might not fit into a single line. In those cases, the text will wrap into two lines and the bubble will follow the last word.

<Example name="fillEquallyMultiline" />

`vertical` makes the Tab Triggers display the `icon` or `title` above the `text`

<Example name="vertical" />

`borderless` removes the border on the bottom of the TriggerList on top of which the underline for the active Trigger is rendered.
This lets you use Tab with a custom Divider implementation or compose them with other components.

<Example name="borderless" />

When Tabs are rendered on top of a dark background, you can set `color` prop to `inherit` to inherit the foreground colors from the parent components.

<Example name="colorInherit" />

Setting the `defaultActiveTabId` prop on the `<Tab>` component activates the Uncontrolled version.

<Example name="uncontrolled" />

Setting the `activeTabId` prop activates the Controlled version of Tab.

<Example name="controlled" />

When the tabs change, the previous tabs get unmounted. If there is a need to keep them mounted (e.g. heavy content) you can use the `keepMounted` prop on `Tab.Panel` components.

### Note

For touch-enabled devices with medium viewports, Tabs are not collapsed into a "more" dropdown menu. Horizontal scrolling with all items visible is enabled for such cases to provide better user expecience.

## Properties

### Tab

| Name               | Type           | Default | Required | Description                                                            |
| ------------------ | -------------- | ------- | -------- | ---------------------------------------------------------------------- |
| vertical           | `boolean`      | -       | false    | Should tab buttons be displayed vertically?                            |
| borderless         | `boolean`      | -       | false    | Remove bottom border?                                                  |
| color              | `inherit`      | -       | false    | Color of the Tabs                                                      |
| fillEqually        | `boolean`      | `false` | false    |
| moreLabel          | `string`       | -       | true     | More label text                                                        |
| activeTabId        | `string`       | -       | false    | Active tab id. Activates controlled version of the component           |
| defaultActiveTabId | `string`       | -       | false    | Default active tab id. Activates uncontrolled version of the component |
| className          | `string`       | -       | false    | Custom root element className                                          |
| attributes         | `object`       | -       | false    | Custom root element attributes                                         |
| onTabChange(id)    | `(id) => void` | -       | false    | Tab change handler                                                     |

### Tab.TriggerList

| Name       | Type        | Default | Required | Description                                |
| ---------- | ----------- | ------- | -------- | ------------------------------------------ |
| children   | `ReactNode` | -       | false    | List of Trigger elements to render         |
| tagName    | `string`    | -       | false    | Custom Triggers wrapper element tag        |
| className  | `string`    | -       | false    | Custom Triggers wrapper element className  |
| attributes | `object`    | -       | false    | Custom Triggers wrapper element attributes |

### Tab.Trigger

| Name           | Type            | Default | Required | Description                                                          |
| -------------- | --------------- | ------- | -------- | -------------------------------------------------------------------- |
| id             | `string`        | -       | true     | ID of the Trigger required to connect to the Panel to render content |
| text           | `string`        | -       | true     | Item text                                                            |
| icon           | `IconComponent` | -       | false    | Item icon                                                            |
| title          | `string`        | -       | false    | Item title                                                           |
| textSlot       | `string`        | -       | false    | Item slot next to the text                                           |
| href           | `string`        | -       | false    | Item href attribute                                                  |
| linkAttributes | `object`        | -       | false    | Item link attributes                                                 |
| content        | `ReactNode`     | -       | false    | Item content                                                         |
| native         | `boolean`       | -       | false    | Item native flag                                                     |

### Tab.Panel

| Name        | Type        | Default | Required | Description                                                                                                    |
| ----------- | ----------- | ------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| id          | `string`    | -       | true     | ID of the Panel correesponding to the respective Trigger                                                       |
| children    | `ReactNode` | -       | false    | Content of the panel                                                                                           |
| tagName     | `string`    | -       | false    | Custom Panel element tag                                                                                       |
| className   | `string`    | -       | false    | Custom Panel element className                                                                                 |
| attributes  | `object`    | -       | false    | Custom Panel element attributes                                                                                |
| keepMounted | `boolean`   | -       | false    | Do not unmount when tab changes                                                                                |
| mixin         | `object`    | -       | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |
