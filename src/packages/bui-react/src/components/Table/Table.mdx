## Usage

```jsx
import { Table } from "@bookingcom/bui-react";
import type { TableProps } from "@bookingcom/bui-react";
```

Tables render based on the data config you pass to them.
Two main data sources for Table are `headings` and `rows` properties which get rendered together by the component.

Every column can have it's alignment and width defined throught the `headings` item properties.

### Advanced layouts

Each table row can store `collapsedContent` which will be shown by clicking on the expand button and with the change of the `expanded` prop. If you need to set the default state of the collapsed content you can use `defaultExpanded`.

You can pass additional `onOpen` and `onClose` handlers in that case to every row to implement custom logic based on the row state.

<Example name="collapsible" />

You can add multi-selection feature to the table by passing a `checkboxName` to the Table and `checkboxValue` to each row.
This will render a checkbox in every row and will let you use `onChange` handler for the Table.

You can also control its value using `value` and `defaultValue` properties.
`value` turns the component controlled mode on, letting you completely control the selection behaviour on the product side.
`defaultValue` turns the component uncontrolled mode on, setting up only the initially rendered selection and then handing over control over it to the browser.

<Example name="selectable" />

Table content is completely customisable which means you can render any other React components in every heading or cell.
This allows you to build more complex features on top of the Table component with minimal effort.

<Example name="sortable" />

## Accessibility

Keep tables simple, avoid having a table within tables with multiple levels of header cells. Keep the sequence of content in a meaningful way.

## Properties

**Table properties:**

| Name          | Type         | Default | Required | Description                                                                                                    |
| ------------- | ------------ | ------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| headings      | `Heading[]`  | -       | false    | Table headings                                                                                                 |
| rows          | `Row[]`      | -       | true     | Table rows                                                                                                     |
| checkboxName  | `string`     | -       | false    | Custom table checkbox name                                                                                     |
| defaultValue  | `string[]`   | -       | false    | Uncontrolled value                                                                                             |
| value         | `string[]`   | -       | false    | Controlled value                                                                                               |
| onChange      | `() => void` | -       | false    | Triggered when indiviual and "all" checkboxes change                                                           |
| onChangeAll   | `() => void` | -       | false    | Triggered when "all" checkbox changes                                                                          |
| stickyHeader  | `boolean`    | -       | false    | Is header sticky?                                                                                              |
| compact       | `boolean`    | -       | false    | Reduce table cell padding?                                                                                     |
| borderless    | `boolean`    | -       | false    | Remove outer borders?                                                                                          |
| overflowFade  | `boolean`    | true    | false    | Add/remove fade when overflowing                                                                               |
| stickyHeader  | `boolean`    | -       | false    | Make table header sticky on large+ viewports                                                                   |
| viewMoreLabel | `string`     | -       | false    | Accessibility label for the expand content button                                                              |
| viewLessLabel | `string`     | -       | false    | Accessibility label for the collapse content button                                                            |
| verticalAlign | `middle`     | -       | false    | Vertical alignment for the cells content                                                                       |
| className     | `string`     | -       | false    | Custom root element className                                                                                  |
| attributes    | `object`     | -       | false    | Custom root element attributes                                                                                 |
| mixin           | `object`     | -       | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |

**Heading properties:**

| Name      | Type                     | Default | Required | Description                      |
| --------- | ------------------------ | ------- | -------- | -------------------------------- |
| content   | `ReactNode`              | -       | true     | Heading content                  |
| width     | `string`                 | -       | false    | Custom width for the column      |
| colspan   | `number`                 | -       | false    |                                  |
| align     | `start`, `end`, `center` | -       | false    | Heading alignment                |
| key       | `string`                 | -       | false    | Custom root element key          |
| className | `string`                 | -       | false    | Custom heading element className |

**Row properties:**

| Name             | Type         | Default | Required | Description                                                            |
| ---------------- | ------------ | ------- | -------- | ---------------------------------------------------------------------- |
| cells            | `Cell[]`     | -       | false    | Row cells                                                              |
| key              | `string`     | -       | false    | Custom root element id                                                 |
| checkboxValue    | `string`     | -       | false    | Row checkbox value                                                     |
| collapsedContent | `ReactNode`  | -       | false    | Collapsed content                                                      |
| onOpen           | `() => void` | -       | false    | Open handler for rows with collapsed content                           |
| onClose          | `() => void` | -       | false    | Close handler for rows with collapsed content                          |
| className        | `string`     | -       | false    | Custom root element className                                          |
| expanded         | `boolean`    | -       | false    | Whether the collapsed content is expanded or not                       |
| defaultExpanded  | `boolean`    | -       | false    | Whether the collapsed content is expanded or not on uncontrolled table |

**Cell properties:**

| Name      | Type        | Default | Required | Description                   |
| --------- | ----------- | ------- | -------- | ----------------------------- |
| content   | `ReactNode` | -       | false    | Cell content                  |
| colspan   | `number`    | -       | false    |                               |
| className | `string`    | -       | false    | Custom root element className |
| key       | `string`    | -       | false    | Custom root element key       |
