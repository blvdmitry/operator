## Usage

```jsx
import { TextVisually } from "@bookingcom/bui-react";
import type { TextVisuallyProps } from "@bookingcom/bui-react";
```

Use text to show text content - titles, paragraphs, string, etc. Notice that Text is responsive throughout breakpoints.

By default Text is rendered as `div` element. You can render any other tag using `tagName` prop. For example, `tagName="span"` to render inline text.

When you have a bidirectional text such as a hotel name that starts with a number the browsers does not always format them correctly in RTL languages. They might move the numbers an punctuations around. In these cases, you can use the `bidirectional` prop to isolate the text content that helps you to format the content correctly in both directions.

## Accessibility

By default, we don't assume in which context text will be used so we're rendering the element as `div`.
However, you can pass a tag to apply the correct `<h\*>` tag and achieve the required accessible ways of navigating the content.

When styling text as a heading (using headline_1, headline_2 or headline_3) a heading tagName (h1 ... h6) should be added to make sure headings are announced with screen-readers also and screen-reader users are provided with the same experience.
Make sure every page only contains a single h1 element and the heading levels are only increasing by one.
