## Usage

```jsx
import { Timeline } from "@bookingcom/bui-react";
import type { TimelineProps, TimelineItemProps } from "@bookingcom/bui-react";
```

Timeline component works with the `children` provided to it and renders them as items.
For the most simple scenario, you can pass other components directly to Timeline and it will wrap every passed child with `Timeline.Item`.
If you also want to pass specific properties to the `Timeline.Item`, you can call it manually as well:

Timeline displays a custom marker for every item but you can replace it with an icon to illustrate the item purpose.

## Properties

**Timeline properties:**

| Name       | Type        | Default | Required | Description                                                                                                    |
| ---------- | ----------- | ------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| children   | `ReactNode` | -       | false    |                                                                                                                |
| className  | `string`    | -       | false    | Custom root element className                                                                                  |
| attributes | `object`    | -       | false    | Custom root element attributes                                                                                 |
| mixin        | `object`    | -       | false    | [Utility Layer](?page=Getting%2520started%252FDevelopment%252FComponents&view=documentation#BUI-Utility-Layer) |

**Timeline.Item properties:**

| Name        | Type              | Default | Required | Description                  |
| ----------- | ----------------- | ------- | -------- | ---------------------------- |
| title       | `ReactNode`       | -       | false    | Item title                   |
| children    | `ReactNode`       | -       | false    | Item content                 |
| marker      | `ComponentType`   | -       | false    | Custom marker icon           |
| markerColor | `TextProps.color` | -       | false    |                              |
| lineVariant | `solid`, `dashed` | `solid` | false    | Connection line display type |
