## Usage

```jsx
import { Tooltip } from "@bookingcom/bui-react";
import type { TooltipProps } from "@bookingcom/bui-react";
```

Here is a basic example of a tooltip

<Example name="default" />

Since tooltips depend on the coordinates of underlying element - it needs a way to hook into its children event handlers.
In order to solve that issue, we're providing a render props pattern for rendering inner components:

```jsx
<Tooltip text="My tooltip text">
  {(attributes) => <button {...attributes}>Tooltip with native button</button>}
</Tooltip>
```

All BUI components support working with attributes, so you can use the same approach with them:

```jsx
<Tooltip text="My tooltip text">
  {(attributes) => <Button text="BUI Button" attributes={attributes} />}
</Tooltip>
```

There may be cases where we want the tooltip to follow the mouse pointer when hovering over the trigger element. Setting the `follow` prop enables this behavior

<Example name="follow" />

### Attaching to custom container

For cases when Tooltip has to be attached to custom container element rather that `body`, pass `containerRef` prop to Tooltip.

> Keep in mind, that custom container element is required to have 'relative', 'fixed' or 'absolute' position set.

## Properties

| Name          | Type                                                                  | Default | Required | Description                                            |
| ------------- | --------------------------------------------------------------------- | ------- | -------- | ------------------------------------------------------ |
| children      | `({ attributes }) => ReactNode`                                       | -       | true     | Tooltip content                                        |
| text          | `string`                                                              | -       | true     | Tooltip hovered text                                   |
| hideArrow     | `boolean`                                                             | -       | false    | Shows a tooltip without the arrow                      |
| follow        | `boolean`                                                             | -       | false    | Tooltip follows cursor                                 |
| position      | `bottom-end`, `bottom`, `bottom-start`, `top-end`, `top`, `top-start` | -       | false    | Position of the tooltip related to the trigger element |
| active        | `boolean`                                                             | -       | false    | Controlled property for toggling visibility            |
| defaultActive | `boolean`                                                             | -       | false    | Uncontrolled property for defining initial visibility  |
| onOpen        | `() => void`                                                          | -       | false    | Tooltip open handler                                   |
| onClose       | `() => void`                                                          | -       | false    | Tooltip close handler                                  |
| containerRef  | `ReactRef`                                                            | -       | false    | Custom container ref                                   |
| zIndex        | `modal`, `drawer`, `number`                                           | -       | false    | Custom z-index value or preset name for tooltip        |
| className     | `string`                                                              | -       | false    | Custom tooltip element className                       |
| attributes    | `object`                                                              | -       | false    | Custom tooltip element attributes                      |
